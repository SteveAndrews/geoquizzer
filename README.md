# GeoQuizzer
GeoQuizzer is a simple and fun way to learn about global geography and the places that shape it.

## Disclaimers
This is a geography learning tool, not a political one. Some geographic areas are politically disputed and are marked accordingly. The information presented reflects the best available data, but may contain errors. No personal information is stored or transmitted to any server except what may be captured by Google Analytics or 3rd party library requests.
